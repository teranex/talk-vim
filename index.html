Modes

Vim is a modal editor.

Modes

  • Insert mode
  • Normal mode
  • Visual mode
  • Command-line mode
  • Operator-pending mode
  • and a few other...

Insert mode

This is the mode where you actually type text. It has a few shortucts, such as C-W to remove the word before the cursor

Normal mode

This is the mode you will use the most. Every key, or combination of keys, is a special command.

Visual mode

Visual mode is an improvement of Vim over the old Vi. It let's you visually select text to act upon.

In fact there are 3 different visual modes

'Regular' visual mode

  • Let's you select text
  • Started by pressing v in normal mode
  • Jump to the other end by pressing o

Linewise visual mode

  • Selects entire lines
  • Started with V (capital v) from normal mode or visual mode

Blockwise visual mode

  • Selects a block of text
  • Started with C-V from normal mode or visual mode
  • This can be very useful. Try, for example, I followed by some text and Esc

Visual mode

When in normal mode, you can always use the gv to reselect the previously selected text in visual mode

Command-line mode

Started with : to enter a command.
When searching (/) you are also in Command-line mode.

If you ever end up in Ex-mode by pressing Q or gQ simply type vi<enter> and carry-on.

Operator-pending mode

This is a very temporary mode which you enter when Vim is waiting for further input. For example after pressing d Vim is in Operator-pending mode while waiting for a motion.

In order to use Vim efficiently, you need to learn it's language. Learn to speak Vim

Motions

Motions are how you move the cursor in Vim.

Jumping to another line

  • gg: jump to first line
